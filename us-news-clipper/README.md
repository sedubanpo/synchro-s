# US Real Estate Daily Clipper

## 개요
이 웹앱은 아래 3개 카테고리의 영문 뉴스를 수집해 카드 형태로 클리핑합니다.

- 미국 모기지 대출 금리
- 미국 부동산 일반
- 미주 한인 부동산

요구사항 반영:
- 카테고리별 분류 표시
- 모기지 금리 기사 필수 포함(미검출 시 백업 기사 자동 삽입)
- 미주 한인 관련 뉴스 포함 체크

## 실행
브라우저에서 `index.html`을 열면 바로 동작합니다.

권장:
- 크롬/엣지 최신 버전
- 네트워크 환경에서 외부 RSS 변환 API(`api.rss2json.com`) 접근 가능해야 함

## 동작 방식
- Google News RSS 검색 결과를 카테고리별로 수집
- 최근 기사 기준 정렬 및 중복 링크 제거
- 30분 자동 갱신 + 수동 갱신 버튼 제공
